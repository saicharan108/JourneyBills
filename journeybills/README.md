**Journey Bills** is a beautifully crafted Android application built with modern **Jetpack Compose** that helps you track your trips, split expenses seamlessly among friends, and manage your travel finances effortlessly. Think of it as a personalized Splitwise tailored specifically for your wanderlust! 🗺️💸

---

## ✨ Features

- 🏕️ **Trip Management:** Organize your adventures by creating and managing multiple trips.
- 🧾 **Expense Tracking:** Effortlessly add expenses, assign who paid, and select who shares the cost.
- ⚖️ **Smart Balances:** Get a crystal-clear breakdown of who owes whom and simplify the process of settling up.
- ☁️ **Cloud Backup & Sync:** Secure your memories and data by exporting/importing backups via **Google Drive** integrations.
- 🎨 **Stunning UI/UX:** Built with the latest **Material Design 3**, featuring dynamic themes, smooth transitions, edge-to-edge layouts, and intuitive controls.
- 🌓 **Dark & Light Mode:** Fully adaptive to your system's theme for optimal viewing day or night.

---

## 🛠️ Built With

The application leverages the latest and greatest in the Android ecosystem:

*   **📱 Language:** [Kotlin](https://kotlinlang.org/) (100%)
*   **🖌️ UI Toolkit:** [Jetpack Compose](https://developer.android.com/jetpack/compose) for a declarative, reactive UI.
*   **🏗️ Architecture:** MVVM (Model-View-ViewModel) paired with Clean Architecture principles.
*   **💾 Local Storage:** [Room Database](https://developer.android.com/training/data-storage/room) for robust offline persistence.
*   **🔗 Reactive Data:** Kotlin Coroutines & Flow for asynchronous tasks and real-time UI updates.
*   **🌐 Integrations:** Google Drive API for secure data backup.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- 💻 Android Studio (Latest stable version recommended)
- 📱 An Android emulator or a physical Android device running API 24 or higher.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/journey-bills.git
   ```
2. **Open the project** in Android Studio.
3. Let Gradle sync the project dependencies.
4. **Run the app** ▶️ on your connected device or emulator.
